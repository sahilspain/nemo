Installation

Getting started with the Cyberpunk CSS Framework is simple. Follow these steps:

Step 1: Include Dependencies

Add these to your HTML <head> section:

<!-- Font Awesome Icons --> <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css"> <!-- Google Fonts --> <link href="https://fonts.googleapis.com/css2?family=Orbitron:wght@400;500;700;900&family=Exo+2:wght@300;400;600&display=swap" rel="stylesheet"> <!-- Cyberpunk CSS Framework --> <link rel="stylesheet" href="cyberpunk.css">

Step 2: Basic HTML Structure

Use this basic template to get started:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Cyberpunk App</title> <!-- Include dependencies as shown above --> </head> <body> <div class="scanlines"></div> <header class="navbar"> <div class="container"> <!-- Your navigation --> </div> </header> <main class="container"> <!-- Your content --> </main> <script src="your-scripts.js"></script> </body> </html>

Tips & Best Practices

  • Always include the .scanlines div for the full cyberpunk effect
  • Use .glitch class on headings for animated glitch effects
  • Combine utility classes for custom styling: class="bg-dark p-3 rounded border-neon-blue"
  • The framework is fully responsive - components adapt to screen size
  • Use Font Awesome icons to enhance the cyberpunk aesthetic
